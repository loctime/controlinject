Extensión de Chrome

Política de Privacidad

ControlBun  ·  Última actualización: mayo 2025

01

Qué hace esta extensión

ControlBun es una extensión de Chrome que automatiza la subida de archivos PDF a la plataforma controldocumentario.com. Lee los requerimientos pendientes del sitio y adjunta los documentos correspondientes sin necesidad de hacerlo manualmente uno por uno.

02

Qué datos se usan

La extensión trabaja con dos tipos de datos:

  • Credenciales de acceso: usuario y contraseña de controldocumentario.com, que el usuario ingresa voluntariamente en las Opciones de la extensión para habilitar el re-login automático.
  • Contenido del sitio: la extensión lee la lista de requerimientos y documentos pendientes que aparecen en controldocumentario.com para poder procesarlos.

🔒 Las credenciales se guardan únicamente en el almacenamiento local de Chrome en tu PC. No se envían a ningún servidor externo ni a los desarrolladores de esta extensión.

03

Qué datos NO se recopilan

Esta extensión no recopila, almacena ni transmite ninguno de los siguientes datos:

  • Datos de salud o médicos
  • Información financiera o bancaria
  • Historial de navegación
  • Ubicación geográfica
  • Comunicaciones personales (correos, mensajes)
  • Actividad del usuario fuera de controldocumentario.com
04

Uso de código remoto

La extensión carga las siguientes librerías externas desde CDN para el procesamiento de PDFs:

  • CDN pdf.js — para renderizar y previsualizar páginas de PDF
  • CDN pdf-lib — para dividir documentos PDF en partes

Estas librerías se cargan únicamente para procesar archivos localmente. No se transmiten datos de usuario a los servidores de dichas librerías.

05

Transferencia de datos a terceros

No vendemos, transferimos ni compartimos datos de usuarios con terceros bajo ningún concepto. Los datos procesados por la extensión se usan exclusivamente para realizar las acciones que el usuario solicita dentro de controldocumentario.com.

06

Permisos de la extensión

La extensión solicita únicamente los permisos mínimos necesarios para funcionar:

  • activeTab — para interactuar con la pestaña activa de controldocumentario.com
  • storage — para guardar credenciales y configuraciones en tu Chrome local
  • scripting — para inyectar el panel de automatización en el sitio
  • Host permission — acceso exclusivo a controldocumentario.com
07

Contacto

Si tenés preguntas sobre esta política de privacidad, podés contactarnos a través del repositorio de la extensión o por los canales habituales de soporte.
